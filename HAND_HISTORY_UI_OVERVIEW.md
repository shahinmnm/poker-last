## Interaction Flow

1. **View Hands List**
   - User clicks history icon on Table page
   - Modal opens with list of recent hands
   - Each hand shows summary info

2. **View Hand Details**
   - User clicks on a hand in the list
   - View transitions to timeline
   - Events grouped by street
   - Chronological order within each street

3. **Navigate Back**
   - User clicks "â† Back to Recent Hands"
   - View returns to list
   - Previous scroll position maintained

4. **View on Stats Page**
   - User navigates to Stats
   - Sees "Recent Hands" section
   - Shows last 5 hands automatically
   - No click needed for initial view

## Performance Metrics

- **Event Logging**: ~5ms per event (async)
- **API Response**: ~50-100ms for 10 hands
- **Timeline Query**: ~20-50ms for single hand
- **UI Render**: ~30ms for timeline view
- **Total UX**: < 200ms from click to display

## Accessibility

- Keyboard navigation: Tab through hands, Enter to open
- Screen reader labels: All actions announced
- Color contrast: WCAG AA compliant
- Touch targets: Minimum 44x44px
- Focus indicators: Visible on all interactive elements
