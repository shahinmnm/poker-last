## Frontend Architecture

### Component Structure
React components for UI elements, TypeScript for type safety, state management with hooks, modular organization.

### API Integration
REST API for data operations, WebSocket for real-time updates, authentication via Telegram, error handling and retry logic.

### Variant Awareness
Dynamic UI based on game variant, variable card displays, variant-specific rules and displays, template-driven behavior.

## Technology Stack

- **React**: UI framework
- **TypeScript**: Type safety
- **Vite**: Build tooling
- **WebSocket**: Real-time communication
