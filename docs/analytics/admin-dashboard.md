# Admin Dashboard

High-level overview of the admin dashboard for analytics consumption.

## Dashboard Purpose

Provides administrative interface for viewing analytics data, exploring insights, monitoring system health, inspecting tables.

## Features

### Real-Time Analytics
Current snapshot view of all tables, live player counts, table statuses, waitlist sizes.

### Historical Data
Hourly statistics browsing, date range filtering, table-specific queries, trend visualization.

### Insights Feed
List of generated insights, severity-based filtering, type-based filtering, time period selection.

### Table Inspection
Detailed table information, current game state, player list, hand history, session metrics.

## Access Control

Restricted to admin users, authentication required, audit logging of admin actions, role-based permissions.

## Data Presentation

Tables and charts for visualization, filtering and sorting options, export capabilities, refresh mechanisms.
